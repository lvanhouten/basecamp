A compact pill for status, counts, or short labels.

```jsx
<Badge tone="success" dot>On track</Badge>
<Badge tone="module">3 active</Badge>
<Badge tone="warning">Due today</Badge>
```

Tones: `neutral`, `brand`, `module` (accent-tinted), `success`, `warning`, `danger`, `solid` (filled). Add `dot` for a leading status dot.
